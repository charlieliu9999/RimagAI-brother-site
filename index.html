系统要求

确保你的系统满足以下最低配置

🍎 macOS

  • macOS 11.0 Big Sur 或更高
  • Apple Silicon (M1/M2/M3) 或 Intel
  • 8GB RAM 推荐

🪟 Windows

  • Windows 10 (64-bit) 或更高
  • x64 或 x86 处理器
  • 8GB RAM 推荐

🐧 Linux

  • Ubuntu 20.04+ / Debian 11+
  • 或其他主流发行版
  • 8GB RAM 推荐

🤖 AI 后端模式（v0.1.1 支持双模式）

  • OpenClaw 模式 - 连接 OpenClaw Gateway (端口 18789)，支持 Agent 编排、多模型 fallback、记忆管理
  • Local Agent 模式 - 内置轻量级 Agent，直接调用 LLM API（OpenAI/Deepseek/Ollama），无需 OpenClaw
  • 💡 在设置面板中配置 OpenClaw 连接，留空则自动使用 Local Agent 模式

🍎 macOS 安装提示

如果打开 .dmg 文件时提示"文件已损坏，无法打开"，这是 macOS Gatekeeper 的安全机制。

解决方法（在终端中执行）：

sudo xattr -cr /Applications/RimagAi\ Brother.app

⚠️ 这是开发阶段的临时解决方案。未来版本将提供 Apple 代码签名和公证，无需此步骤。
