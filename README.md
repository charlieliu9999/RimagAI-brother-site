# RimagAi Brother 产品页面

## 📁 文件说明

- `index.html` - 产品主页（单页应用）
- `assets/` - 图片与静态资源

## 🚀 部署方式

### 方式 1：GitHub Pages

1. 将仓库推送到 GitHub
2. 在仓库设置中启用 GitHub Pages
3. 选择 `main` 分支的 `root` 目录
4. 访问 `https://charlieliu9999.github.io/RimagAI-brother-site/`

### 方式 2：Vercel / Netlify

直接拖拽本项目目录到平台即可部署。

### 方式 3：本地预览

```bash
python3 -m http.server 8080
# 访问 http://localhost:8080
```
